This is a simulation of the wave equation ∂2u/∂t2 = c2(∂2u/∂x2+∂2u/∂y2) across a circular 2D manifold. Use the mouse to create waves.

This compares a standard Javascript implementation against three different WebAssembly modules:

  • C code compiled by Enscripten
  • JS-like code compiled by Walt
  • TypeScript-like code compiled by AssemblyScript
